# TechWave Podcast Website

<img src="./TechWave.png">

## Overview
**TechWave** is a modern and responsive podcast website designed to showcase episodes, hosts, and subscription options. It offers an engaging interface for tech enthusiasts to stay updated with tech insights, productivity hacks, and personal growth tips.

---

## Features
- Fully responsive design using **Tailwind CSS**.
- Google Fonts integration for modern typography (`Inter` font).
- Font Awesome icons for navigation and buttons.
- Header with navigation menu and subscribe button.
- Banner section highlighting the latest episode.
- About section describing the podcast.
- Statistics section displaying key numbers (listeners, episodes, ratings, experts).
- “Why Choose TechWave” section highlighting the podcast features.
- Featured Episodes section with embedded YouTube videos.
- Host section with social media links.
- Footer section linking major podcast platforms (Spotify, Apple Podcasts, YouTube, Twitter).

---

## Technologies Used
- **HTML5** – Markup language for structure.
- **CSS3 / Tailwind CSS** – Styling and responsive design.
- **Google Fonts** – Inter font for modern typography.
- **Font Awesome** – Icons for UI elements.
- **YouTube Embed** – Featured episodes section.
- **Images & Icons** – Stored in `./assets/` folder.
